# C++ DSA Learning Project

This repository contains starter code and resources for a three‑month curriculum to learn Data Structures & Algorithms in C++.

- `main.cpp` – a minimal program that prints "Hello, DSA!".
- `hello_world.cpp` – prints a simple "Hello World" message.
- `calculator.cpp` – a command-line calculator performing basic arithmetic.
- `guessing_game.cpp` – a game to guess a randomly generated secret number.
- `prime_checker.cpp` – an optimized program that checks if a number is prime.
- `README.md` – this file.
- `.gitignore` – typical C++ ignores.

Feel free to add more source files as you progress through the roadmap.
